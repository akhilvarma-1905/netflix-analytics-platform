# 🎬 Netflix Analytics and Recommendation System

An end-to-end Data Analytics and Web Application project built using Python, PostgreSQL, Streamlit, Power BI, and Zoho Analytics.

This project analyzes Netflix movies and TV shows to generate interactive dashboards, business insights, and smart content recommendations.

## 🚀 Features

* Interactive Analytics Dashboard
* Movies & TV Shows Analysis
* Genre & Rating Insights
* Country-wise Map Visualizations
* KPI Metrics & Trend Analysis
* Smart Recommendation System
* PostgreSQL Database Integration
* Responsive Netflix-inspired Web UI

## 🛠️ Technologies Used

Python • Pandas • PostgreSQL • Streamlit • Power BI • Zoho Analytics • Plotly • Machine Learning

## 📊 Insights Generated

* Most popular genres
* Content distribution by country
* Ratings analysis
* Release year trends
* Movies vs TV Shows comparison
* Intelligent content recommendations

## 🌐 Project Goal

Transform raw Netflix data into meaningful insights and recommendations through analytics, visualization, and interactive web technologies.
